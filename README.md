# GeoFS-Aircraft-Physics-Fixes-Addon
Fixes the flap performance of GeoFS Aircraft
This fix applies to all aircraft
- This fix is optimized for spice9 aircraft, but works on others as well.
- The A330-200 and A340-300 are the only ones tested not working
- Aircraft with working flaps do not fly differently
## How to install

Click on the file that says "Userscript.js"

Copy the contents of it, and paste it into your script loader.

Additionally you can also paste it into your browser console
